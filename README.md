# Rede peer-to-peer
Trabalho de Sistemas distribuídos



## Bibliotecas

 - Pyro5
 - uuid
 - time
 - threading
 - os
 
 ## Execução
 
 1 - É necessário chamar o servidor de nomes
 
 >~/.local/bin/pyro5-ns


 2 - Executar o cliente/servidor
 > python Cliente_Pyro.py


## Limitações
